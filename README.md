# Kali DevSec Workstation

A reproducible Kali Linux workstation for:

- Software Development
- DevOps
- Cybersecurity
- Networking
- Virtualization

## Features

- Docker Engine
- Docker Compose
- Git & GitHub CLI
- Node.js (NVM)
- pnpm
- Bun
- Python Development
- Java 21
- PostgreSQL
- VirtualBox
- Cursor
- Visual Studio Code
- Cisco Packet Tracer
- GNS3

## Directory Structure

Projects are organized under:

~/Projects
├── MOH
├── Personal
├── OpenSource
└── Learning

./bootstrap.sh
